# Examples

Sample output files generated by pr-scout for a real-world review run.

```
outputs/DASH-77-pr-19/
├── briefing.md            — story verdict, PR narrative walkthrough, compliance matrix, findings
├── review-comments.json   — proposed inline comments with diff positions, ready to post
└── manual-todo.md         — findings that couldn't be anchored to a diff line, access failures
```

These files mirror the full output contract under `.ai/pr-scout/outputs/<run-id>/`.
Nothing is posted to the PR until you review and confirm the comment list in chat.
